````markdown
## What is StudyHive?

StudyHive is a **full-stack AI academic operating system** powered by **ARIA** (Adaptive Reasoning & Intelligence Architecture) вЂ” built on Google Gemini 2.5 Flash.

ARIA builds a cognitive model of each student over time:

```
Student answers questions
         в†“
ARIA records: concept, difficulty, time spent, correct/wrong
         в†“
Updates: Knowledge Graph В· Error DNA В· Forgetting Curve В· XP
         в†“
Dashboard shows: real accuracy В· real weak topics В· real predictions
         в†“
ARIA personalizes: every question В· every explanation В· every plan
```

This is **Notion + Discord + Khan Academy + ChatGPT + Duolingo** вЂ” built specifically and obsessively for the Indian competitive exam ecosystem.

```typescript
{
  userId: ObjectId
  knowledgeGraph: [{
    concept: string
````
