## 🔥 The Problem

Most ed-tech is passive. Watch video. Take test. Repeat. **Dead.**

10 million+ students prepare for JEE, NEET, UPSC every year. **90% fail** — not because they aren't smart, but because they:

- Study the wrong topics at the wrong time
- Don't know *why* they keep making the same mistakes
- Have no feedback loop between effort and actual improvement
- Spend ₹2-5 lakh on coaching that still can't personalize to them

**StudyHive 2.0 fixes all of this.**

---

## 🧠 What is StudyHive?

StudyHive is a **full-stack AI academic operating system** powered by **ARIA** — Adaptive Reasoning & Intelligence Architecture — built on **Google Gemini 2.5 Flash**.
Student answers questions
↓
ARIA records: concept · difficulty · time spent · correct/wrong
↓
Updates: Knowledge Graph · Error DNA · Forgetting Curve · XP · Streak
↓
Dashboard shows: real accuracy · real weak topics · real predictions
↓
ARIA personalizes: every question · every plan · every explanation

This is **Notion + Discord + Khan Academy + ChatGPT + Duolingo** — built specifically and obsessively for the Indian exam ecosystem.

## ⚡ Core Features

### 🤖 ARIA — Adaptive AI Coach
| Feature | Description |
|---------|-------------|
| **Knowledge Graph** | Live node map of every concept, colored by mastery level |
| **Error DNA** | Names and patterns your specific recurring mistakes |
| **Peak Window** | Identifies your best 2-3 hour cognitive performance window |
| **Forgetting Curve** | SM-2 spaced repetition, recalibrated to YOUR forgetting rate |
| **Score Prediction** | Daily percentile projection with confidence intervals |
| **Daily Brief** | Personalized morning briefing — what to study, why, how long |
| **4 Coaching Modes** | Strict · Supportive · Socratic · Strategist |
| **Crash Plan** | Emergency revision plan — exam in 48 hours? ARIA handles it |

### ⚔️ Battle Arena
| Mode | Description |
|------|-------------|
| **1v1 Lightning** | Real opponent · 10 questions · 90 seconds each · ELO rating |
| **Daily Blitz** | 5 questions · Community-wide · Same for everyone (like Wordle) |
| **The Gauntlet** | 100 questions · Adaptive difficulty · Global leaderboard |
| **Boss Battles** | Weekly ultra-hard concept · First 100 correct get Boss Slayer badge |
| **Hive Wars** | 5v5 team battles · Win together or lose together |

### 📊 Cognitive Analytics
- 90-day performance curves per exam subject
- **Error DNA Analysis** — Sign Flip Syndrome · Formula Recall Lag · Unit Conversion Skip
- **Score Prediction Engine** — confidence intervals + scenario modeling
- **Subject Balance Radar** — visual mastery across all exam subjects
- **Autopsy Report** — time heatmap, careless vs conceptual breakdown after every mock

### 📚 DeepStudy — AI Learning Tools
- **Smart Notes Editor** — ARIA auto-tags concepts and flags knowledge gaps
- **Concept Crystallizer** — paste any text → exam-ready structured summary
- **YouTube → Notes** — paste lecture URL → timestamped structured notes
- **PDF Annihilator** — upload NCERT → exam-ready summary + practice questions
- **Mind Map Generator** — type chapter name → interactive concept map
- **Formula Vault** — every formula linked to question types + memory tricks
- **Bidirectional Flashcards** — SM-2 spaced repetition, auto-generated from notes

### 🏠 Hive Rooms
| Type | Description |
|------|-------------|
| **Silent Hive** | Pomodoro focus · Ambient sounds · Peer pressure without chat |
| **Discussion Hive** | Live chat + AI whiteboard + voice |
| **Lecture Hive** | One teaches, rest attend · ARIA auto-summarizes recordings |
| **Battle Hive** | Same question for everyone · Fastest correct answer wins |
| **Accountability Hive** | Daily check-in · State goals · Community witnesses |

### 📅 Smart Scheduler
- ARIA builds **day-by-day, hour-by-hour** study plan for your entire prep
- Adjusts **every day** based on what you actually did yesterday
- Missed a session? Redistributes the content, never drops it
- **Rescue Mode** — falling behind triggers a compressed catch-up plan
- Google Calendar sync · WhatsApp/Telegram reminders

### 🏆 Hive Identity
- **Ranks:** Apprentice → Scholar → Thinker → Innovator → Architect → Genius → Einstein
- **Skill Crystals** — earned for every mastered concept, collected into constellations
- **Rivalry System** — challenge friends to month-long competitions with real stakes
- **Legacy** — your anonymized prep data helps future students after you crack the exam

## 🏗️ How It Was Built
Google Stitch  ──MCP──▶  Antigravity IDE  ──▶  GitHub  ──▶  Vercel + Railway
(UI Design)              (AI Coding)         (Code)       (Production)

1. Generated 6 high-fidelity screens in **Google Stitch** via Vibe Design prompts
2. Connected Stitch to **Antigravity** via MCP — fetched Design DNA (34 colors)
3. AI agent converted Stitch HTML → React components
4. Upgraded every element with **ShadCN** replacements
5. Added complete **GSAP** animation layer
6. Built **Node.js backend** with Gemini AI integration
7. Implemented full **auth system** + **MongoDB** schemas
8. Deployed pipeline ready for **Vercel + Railway**
